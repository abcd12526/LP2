STEP 4 - GOOGLE CLOUD CONSOLE - CREATE PROJECT
1. Go to https://console.cloud.google.com
2. Click the three lines (=) top left -> Google Cloud Overview -> Dashboard
3. Click the project dropdown at the top -> New Project
4. Enter Project Name -> Click Create
5. Click the Bell icon (notifications) at top right
6. Click Select Project


STEP 5 - ENABLE APP ENGINE ADMIN API
1. In the search bar type: App Engine Admin API
2. Click on it -> Click Enable


STEP 6 - ACTIVATE CLOUD SHELL
1. Click the Cloud Shell icon (>_) at top right
2. Cloud Shell terminal opens at the bottom
